<h2>A fully interactive dashboard covering 14 government exams with their details, in an interactive and visual manner.</h2>





<h2>What's in it:</h2>

<ul>

<li>5 quick-stat chips at the top for instant orientation (total exams, bodies, breakdown by category). </li>

<li>Filter buttons — click State WB, Central Govt., or Banking (IBPS) to instantly hide/show only the relevant exams. </li>

<li><h3>Two view modes:</h3>

<ol>

<li>Cards view — each exam is a card with key facts (qualification, age, negative marking, pay) visible at a glance, with a "▼ Exam Details" toggle for the full paper-wise breakdown </li>

<li>Comparison Table — all 14 exams side-by-side in one table; the filter buttons work here too. </li>

</ol>

</li>



<li>Color coding throughout: 🔵 Blue = State WB, 🟠 Orange = Central Govt., 🟢 Green = Banking </li>

</ul>
